AI 언어 모델 발전 과정

From Seq2Seq to ChatGPT

칠판의 내용을 기반으로 재구성한 프레젠테이션입니다.

1. Sequence-to-Sequence (Seq2Seq)

기계 번역을 위해 등장한 초기 모델 구조입니다. 문장 전체를 하나의 벡터로 압축하고 다시 문장으로 풀어냅니다.

입력 문장
(Source)
→
Encoder
→
Context
Vector
→
Decoder
→
출력 문장
(Target)

한계점:

  • 긴 문장에서 정보 손실(Information Bottleneck)이 발생할 수 있습니다.
  • Hallucination (환각): 사실과 다르거나 관련 없는 내용을 생성할 수 있습니다.

2. 인코더(Encoder) 기반 모델: BERT

문장의 '의미'와 '문맥'을 이해하는 데 중점을 둔 모델입니다. (by Google)

사전 학습 (Pre-training)

방대한 텍스트 데이터로 언어 자체의 패턴과 구조를 미리 학습합니다.

양방향성 (Bidirectional)

단어의 의미를 파악하기 위해 문장의 앞뒤 문맥을 모두 동시에 고려합니다.

활용 방식: 미세 조정 (Fine-Tuning)

사전 학습된 모델을 특정 과제(예: 감성 분석)에 맞게 소량의 데이터로 추가 학습하여 사용합니다.

3. 디코더(Decoder) 기반 모델: GPT 시리즈

다음 단어를 예측하며 '문장을 생성'하는 데 특화된 모델입니다. (by OpenAI)

GPT-1

지도 학습 방식을 활용한 초기 모델

GPT-2

모델과 데이터 크기를 키워 사전 학습의 성능을 극대화

GPT-3

1,750억 개 파라미터로 모델의 생성 능력을 비약적으로 향상

ChatGPT의 핵심 기술: 정렬(Alignment)

모델이 단순히 글을 잘 쓰는 것을 넘어, 인간의 '의도'와 '가치'에 맞게 행동하도록 만드는 과정입니다.

RLHF (인간 피드백을 통한 강화학습)

1. 사람이 직접 좋은 답변과 나쁜 답변을 평가하여 '보상 모델'을 만듭니다.

2. 언어 모델이 더 좋은 평가(보상)를 받는 방향으로 답변을 생성하도록 강화학습을 통해 스스로를 개선합니다.

SFT (Supervised Fine-Tuning)

사람이 작성한 고품질의 질문과 답변 데이터를 통해 모델이 지시를 잘 따르도록 훈련시킵니다.

요약 및 비교

모델 주요 목적 구조 특징
Seq2Seq 번역, 요약 Encoder-Decoder 정보 압축 및 변환
BERT 문맥 이해, 분류 Encoder (Transformer) 양방향성, 사전학습
