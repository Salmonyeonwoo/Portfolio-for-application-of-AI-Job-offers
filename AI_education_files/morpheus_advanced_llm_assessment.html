Morpheus 고급 파이프라인 및 대규모 모델 평가 요약

DeepSpeed와 RouterStage를 활용한 LLM 메모리 최적화 및 동적 평가 파이프라인 구축

1. 프로젝트 개요 및 핵심 목표

Morpheus의 비선형 구조와 **DeepSpeed** 프레임워크를 통합하여, **대규모 언어 모델(LLM) 평가**를 위한 메모리 효율적인 솔루션을 구현했습니다. 이는 복잡하고 리소스 집약적인 AI 파이프라인을 설계하고 **모델 리소스를 최적화**하는 능력을 입증합니다.

핵심 기술 역량

DeepSpeed Optimization LLM Assessment RouterStage / Non-Linear Pipeline Stage-graphs Modularity

LLM 평가 파이프라인 구조

2. 기술적 도전 및 LLM 메모리 최적화

2-1. 도전: LLM 평가를 위한 메모리 효율성 확보

MinGPT, Vision Transformers 같은 대규모 모델을 GPU 환경에서 안정적으로 평가하기 위해서는 **방대한 GPU 메모리 관리**가 필수적이었습니다.

✅ 해결: DeepSpeed 최적화 기술 적용

**DeepSpeed**를 활용하여 모델을 포팅하고 다음과 같은 최적화 전략을 적용했습니다.

  • **활성화 체크포인팅 (Activation Checkpointing):** 메모리 재계산을 통해 메모리 사용량을 절감.
  • **ZeRo 리던던시 옵티마이저:** 모델 상태를 분산 저장하여 메모리 오버헤드를 획기적으로 감소.

2-2. 유연한 파이프라인 구조 설계

비선형 파이프라인(`Pipeline` 클래스)과 **`RouterStage`**를 통합하여, 데이터의 조건에 따라 **LLM 평가 브랜치**와 **일반 추론 브랜치**로 메시지를 동적으로 분기하는 유연한 구조를 설계했습니다.

2-3. 모듈화 및 재사용성 확보

복잡한 LLM 전처리/후처리 로직을 단일 커스텀 스테이지 내부에 MRC 노드들을 연결하는 **스테이지 그래프(Stage-graphs)** 형태로 모듈화하여, 코드의 재사용성을 높이고 유지보수를 용이하게 했습니다.
