3. 예측 유지보수: 오토인코더 기반 이상 탐지 요약

재구성 오류 기반의 오토인코더(AE) 딥러닝 이상 탐지 기법 구현

ML

프로젝트 개요

시계열 데이터에서 **이상 패턴(Anomalies)**을 탐지하기 위해 **오토인코더(Autoencoder, AE)** 딥러닝 모델을 사용했습니다. 모델이 정상 데이터만을 학습하도록 훈련시킨 후, 정상 패턴에서 벗어나는 데이터를 이상으로 분류하는 기법을 구현했습니다.

⚙️ 주요 기술 및 역할

  • 기술: Autoencoder(인코더/디코더), 이상 탐지(Anomaly Detection), 재구성 오류(Reconstruction Error).
  • 이상 탐지 방법론: 오토인코더가 **정상 데이터만을 학습**하도록 훈련 데이터셋을 구성하고, 복원된 데이터와 원본 데이터 간의 **재구성 오류(Reconstruction Error)** 임계치를 설정하여 이상 여부를 판단하는 논리를 구현했습니다.
  • GPU 훈련 최적화: GPU를 사용하여 대용량 데이터 기반의 오토인코더 모델을 효율적으로 훈련했습니다.

🛠️ 주요 구현 내용

  • 훈련 데이터 구성: 훈련 데이터셋에서 **정상(Non-defective) 샘플**만을 필터링하여 오토인코더 입력으로 사용했습니다.
  • 모델 구조: 입력층, 인코더, 잠재 공간(Latent Space), 디코더, 출력층으로 구성된 네트워크를 설계했습니다.
  • 이상 탐지: 훈련된 모델에 테스트 데이터를 입력했을 때 발생하는 **재구성 오류(MSE)**가 설정된 임계치를 초과할 경우, 해당 데이터를 '이상(Anomaly)'으로 분류했습니다.

📌 주요 코드 로직 (오토인코더 훈련 데이터 준비 추상화)

# 'failure' 라벨이 0인 정상 샘플만 추출하여 훈련 데이터셋 구성
x_train_normal = x_train[df_train['failure'] == 0]

# 오토인코더는 입력(X)을 출력(X)으로 복원하도록 학습 (입력 == 목표)
history = autoencoder.fit(
    x_train_normal, 
    x_train_normal, 
    epochs=EPOCHS,
    batch_size=BATCH_SIZE,
    validation_data=(x_test, x_test)
