CrewAI - 주식 분석팀 요약

다중 LLM 에이전트(Multi-Agent)가 협력하여 주식 분석 리포트를 생성하는 CrewAI 프로젝트

API

프로젝트 개요

LangChain 기반의 **CrewAI 프레임워크**를 활용하여, 각기 다른 역할을 맡은 **다중 AI 에이전트(Multi-Agent)**들이 협업하여 주식 시장 분석 및 투자 전략 리포트를 생성하는 시스템을 구현했습니다. 이는 복잡한 실무 태스크를 LLM 기반으로 자동화하는 능력을 입증합니다.

⚙️ 주요 기술 및 역할

  • 기술: CrewAI, LLM (OpenAI/Gemini API), LangChain.
  • 에이전트 역할 정의: '시장 분석가(Market Analyst)', '투자 전략가(Strategy Planner)', '최종 리포터(Reporter)' 등 전문화된 역할을 가진 에이전트들을 정의했습니다.
  • 협업 태스크 설계: 에이전트 간의 정보 교환 및 순차적 작업을 정의하는 **태스크(Task)**와 **프로세스(Process)**를 설계하여, 최종적으로 통합된 리포트를 생성하도록 했습니다.

🛠️ 주요 구현 내용

  • Tool 연동: 에이전트가 실시간 정보를 얻을 수 있도록 Google Search 등의 외부 **도구(Tool)**를 LLM에 연결했습니다.
  • 순차적 프로세스: 시장 분석가의 결과를 전략가가 받아 전략을 수립하는 **순차적 프로세스**를 CrewAI에서 설정했습니다.

📌 CrewAI 에이전트 구조

# Agent Definition Example
Agent(
    role='Market Analyst',
    goal='Identify key trends and risks in the stock market.',
    backstory='Expert in financial data analysis.',
    tools=[SearchTool]
)

# Crew Definition Example
Crew(
    agents=[analyst, strategist, reporter],
    tasks=[task1, task2, task3],
    process=Process.sequential
)
