[Morpheus] 커스텀 스테이지 기본기 및 메시지 업데이트

Morpheus 커스텀 스테이지 정의 및 MessageMeta 페이로드(cuDF) 안전 업데이트 로직 구현

Morpheus

프로젝트 개요

Morpheus 파이프라인의 유연성을 위해 필수적인 **사용자 정의 스테이지(Custom Stage)** 작성 방법을 학습하고, 데이터 메시지(MessageMeta 또는 ControlMessage)의 내부 데이터(cuDF DataFrame)를 안전하게 읽고 수정하는 로직을 구현했습니다.

⚙️ 주요 기술 및 역할

  • 기술: Morpheus SinglePortStage, MessageMeta.mutable_dataframe(), GpuAndCpuMixin.
  • Custom Stage 정의: `SinglePortStage`를 상속받아 Morpheus 파이프라인에 통합될 커스텀 클래스의 기본 구조(name, on_data)를 확립했습니다.
  • 안전한 데이터 업데이트: `with message.mutable_dataframe() as df:` 컨텍스트 관리자를 사용하여 GPU 메모리에 있는 cuDF 데이터를 안전하게 수정함으로써, 멀티 스레드 환경에서 발생할 수 있는 **경쟁 조건(Race Condition)**을 방지했습니다.

🛠️ 주요 구현 내용

  • Passthrough Stage: 입력 메시지를 변경 없이 그대로 반환하는 가장 기본적인 `Passthrough Stage`를 구현하여 데이터 흐름을 확인했습니다.
  • 메시지 페이로드 수정: 사용자 인증 로그 데이터에서 'user' 컬럼의 값을 **대문자(Uppercase)**로 일괄 변경하는 데이터 변환 로직을 `on_data` 메서드 내에 적용했습니다.

📌 주요 코드 로직 (메시지 페이로드 업데이트 추상화)

from morpheus.pipeline.single_port_stage import SinglePortStage
from morpheus.messages import MessageMeta

# 메시지 페이로드(DataFrame)를 수정하는 커스텀 스테이지
class UpdateMessageStage(SinglePortStage):
    def on_data(self, message: MessageMeta) -> MessageMeta:
        # mutable_dataframe() 컨텍스트 관리자를 사용하여 안전하게 데이터 수정
        with message.mutable_dataframe() as df:
            # 예시: 'user' 컬럼의 모든 값을 대문자로 변경
            if 'user' in df.columns:
                df['user'] = df['user'].str.upper()
                
        return message
