Project 1: RIVA ASR 파이프라인 및 NGC 리소스 관리

NGC를 통한 AI 리소스 관리 및 서버-클라이언트 아키텍처 기반 실시간 음성 인식 시스템(ASR) 구축.

⚙️ 핵심 기술 및 역할

  • • **NGC CLI & NGC API Key**: 독점 AI 모델 및 리소스 다운로드 인증 관리.
  • • **Riva gRPC API**: 로컬 컨테이너(localhost:50051)에 ASR 추론 요청 전송.
  • • **NeMo Model**: End-to-End ASR 구현 및 ITN 후처리로 정확도 최적화.
  • • **Docker 컨테이너**: Riva 서버 배포 및 운영 환경 구축.

🌟 실무 활용 분야

  • ✓ **대화형 AI 서비스**: 콜센터 자동화, 실시간 회의록 작성 등 대규모 실시간 음성 처리 시스템 구축.
  • ✓ **MLOps 파이프라인**: 엔터프라이즈 환경에서 모델 및 데이터셋을 안전하게 배포하고 관리.

# Riva 클라이언트 핵심 로직 (Python 추상화)

auth = riva.client.Auth(uri='localhost:50051')
riva_asr = riva.client.ASRService(auth)
response = riva_asr.recognize(audio_data, config)
# -> 실시간 음성 데이터 변환 및 전송

Project 2: RIVA QA 시스템 통합 및 평가 (가상 비서)

ASR, QA, TTS를 통합하여 Speech-to-Speech 기반의 End-to-End 대화형 AI 어시스턴트 구축.

⚙️ 핵심 기술 및 역할

  • • **ASR → QA → TTS 파이프라인**: 음성 입력부터 음성 답변까지 순차적 연동 구현.
  • • **Riva QA Service**: BERT 기반 Extractive QA 모델을 활용하여 지식 문서 내 정답 추출.
  • • **Riva TTS Service**: QA 답변 텍스트를 고품질 합성 음성으로 변환.

🌟 실무 활용 분야

  • ✓ **전문 지식 기반 QA**: 콜센터, 교육용 챗봇 등 지식 영역 내에서 정확한 정답 스팬 추출.
  • ✓ **음성 인터페이스**: 내비게이션, 키오스크 등 음성 입출력이 필수적인 서비스.

# End-to-End 파이프라인 (추상화)

transcript = asr_predict(audio_sample)
dm_answer = dm_predict(transcript, knowledge_context) # 정답 추출
synth_audio = tts_predict(dm_answer)
