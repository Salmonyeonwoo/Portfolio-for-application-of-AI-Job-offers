[Morpheus] 고급 파이프라인 및 LLM 평가 솔루션 요약

비선형 흐름, 커스텀 스테이지 내 그래프 구축 및 DeepSpeed LLM 평가 통합

Morpheus LLM

프로젝트 개요

Morpheus의 고급 기능들을 통합적으로 활용하여 **비선형 데이터 흐름**, **조건부 라우팅** 등을 구현하고, 나아가 **DeepSpeed** 프레임워크를 이용한 **대규모 언어 모델(LLM) 평가 솔루션**까지 구현했습니다. 이는 복잡한 엔터프라이즈급 AI 파이프라인 설계 및 운영 능력을 입증합니다.

⚙️ 주요 기술 및 역할

  • 기술: Morpheus `Pipeline` (비선형), `RouterStage`, MRC Builder, DeepSpeed (LLM Assessment 추론).
  • 비선형 및 라우팅: `Pipeline` 클래스와 `RouterStage`의 `key_fn`을 사용하여 메시지 페이로드 조건에 따른 **동적 분기**를 구현했습니다.
  • 커스텀 스테이지 그래프: 단일 커스텀 스테이지 내부에 MRC 노드들을 병렬 또는 직렬로 연결하여 복잡한 로직을 모듈화하는 방법을 습득했습니다.
  • 대규모 모델 통합: `minGPT`, `Vision Transformers`와 같은 대규모 모델을 **DeepSpeed**로 최적화하고, `Assessment` 추론을 위한 복잡한 딥러닝 트레이닝 파이프라인을 구축했습니다.

🛠️ 주요 구현 내용 (LLM Assessment 통합)

  • DeepSpeed 최적화: LLM 훈련 시 메모리 효율성을 극대화하는 **활성화 체크포인팅** 및 **ZeRo 리던던시 옵티마이저** 기술을 적용했습니다.
  • 다단계 PCAP 분석: 복잡한 PCAP 데이터 분석을 여러 커스텀 스테이지로 분리하여 선형 파이프라인으로 구성하는 다단계 전처리 능력을 시연했습니다.
  • 라우팅 유연성: 메시지에 태그된 'route' 메타데이터를 기반으로 'nvsmi\_route'와 'pcap\_route'를 분기하는 라우팅 함수를 구현했습니다.

📌 주요 코드 로직 (비선형 라우팅 추상화)

from morpheus.stages.general.router_stage import RouterStage
from morpheus.messages import MessageMeta

# 라우팅 키 함수: 메시지 메타데이터 상태를 기반으로 분기 결정
def router_key_fn(message: MessageMeta) -> str:
    # 캡스톤 프로젝트에서 NVSMI와 PCAP을 분기하는 데 사용된 로직과 유사
    route_key = message.get_metadata('route')
    if route_key == 'nvsmi_data':
        return "nvsmi_route"
    else:
        return "pcap_route" # 기본값 또는 다른 경로로 처리
