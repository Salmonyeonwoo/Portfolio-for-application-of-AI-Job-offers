1. 예측 유지보수: GPU XGBoost 요약

NVIDIA RAPIDS cuML을 활용한 GPU 가속 XGBoost 시계열 분류 프로젝트

GPU ML

프로젝트 개요

하드 드라이브 SMART 데이터를 사용하여 고장/정상 상태를 분류하는 예측 유지보수(Predictive Maintenance) 모델을 구축했습니다. 특히, **NVIDIA RAPIDS** 라이브러리를 활용하여 **XGBoost 모델의 학습 및 테스트 과정**을 GPU로 가속하여 성능을 최적화했습니다.

⚙️ 주요 기술 및 역할

  • 기술: NVIDIA RAPIDS (cuDF, cuML), XGBoost Classifier, 시계열 데이터 전처리.
  • 클래스 불균형 해소: 고장 데이터가 극히 일부인 **클래스 불균형(Imbalanced Classes)** 문제를 해결하기 위해 데이터 오버샘플링 또는 가중치 부여 기법을 적용했습니다.
  • GPU 가속: cuDF 데이터 구조를 사용하여 XGBoost 모델을 훈련함으로써, 대규모 데이터셋에서도 CPU 대비 월등히 빠른 속도로 모델을 훈련하고 평가했습니다.
  • 데이터 준비 및 정제: 불필요한 컬럼 제거, 누락된 값(NaN/Null) 처리 등 데이터 큐레이션 과정을 수행했습니다.

🛠️ 주요 구현 내용

  • cuDF 로드: Pandas 대신 cuDF를 사용하여 데이터를 GPU 메모리에 직접 로드하고, 모든 전처리 작업을 GPU 상에서 처리했습니다.
  • 특징 공학(Feature Engineering): 고장 예측을 위해 필요한 핵심 SMART 지표만을 선별하고 정규화(Normalization)했습니다.
  • 모델 훈련: cuML 라이브러리의 GPU 기반 XGBoost 모델을 사용하여 분류 모델을 훈련하고 성능 지표를 검증했습니다.
