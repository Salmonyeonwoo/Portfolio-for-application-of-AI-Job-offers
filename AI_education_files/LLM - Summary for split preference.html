4. LLM 선호도 예측 분류 모델 요약

LLM 응답에 대한 사용자 선호도를 예측하는 다중 분류(Multi-Class Classification) 모델 개발

AI

프로젝트 개요

사용자의 응답 데이터를 기반으로 두 가지 LLM 모델(Model A vs Model B) 중 어떤 응답을 선호할지 예측하는 **다중 분류(Multi-Class Classification) 모델**을 개발했습니다. 이는 LLM의 품질 관리(QA) 및 모델 개선 방향 설정에 중요한 피드백 메커니즘을 제공합니다.

⚙️ 사용 기술 및 역할

  • 기술: Python, Pandas, Scikit-learn (MinMaxScaler, MultinomialNB, SVC, RandomForest), TF-IDF.
  • 고급 특징 엔지니어링: 텍스트 응답에 **TF-IDF 벡터화**를 적용하여 텍스트 특징을 추출하고, 수치형 특징(토큰 수, 구두점 수)과 결합하여 예측력을 높였습니다.
  • 스케일링 문제 해결: Multinomial Naive Bayes(MNB) 모델 적용 시, 입력 특징이 음수여서 발생하는 학습 실패 문제를 **MinMaxScaler**를 통해 해결했습니다.

🛠️ 주요 구현 내용

  • 특징 공간 확장: 텍스트 내용과 응답 길이/구조를 모두 고려하는 통합 특징 행렬을 생성했습니다.
  • 모델 비교: MNB, SVC, RandomForestClassifier를 적용하고 Log Loss 지표로 성능을 비교 분석했습니다. (MNB에서 Log Loss 0.0119 달성)
